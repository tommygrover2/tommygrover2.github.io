Using tidyverse functions and chaining or piping to complete the desired manipulations to the data set. First using the select function to include all variables other than eccentricity and roundness, then using the rename function to shorten the names for the major and minor axis length variables. From here I used the filter function in combination with the %in% operator to only include observations in which the class variable was one of the ones in a specific list. Continuing, I use a mutate function to add the average length variable, as well as a new categorical variable that relates to aspect ratio. Finally, I finish by ordering the rows by convex area in descending order and calling my new object.

new_beans_data <- beans_data %>%
  select(-Eccentricity, -Roundness) %>%
  rename(LengthMaj = MajorAxisLength, LengthMin = MinorAxisLength) %>%
  filter(Class %in% c("DERMASON", "SIRA", "CALI", "HOROZ", "BOMBAY")) %>%
  mutate(AvgLength = (LengthMaj + LengthMin) / 2,
         AspectRatioCategory = case_when(
           AspectRatio > 1.650262 ~ "Largest",
           AspectRatio > 1.513478 ~ "Medium",
           TRUE ~ "Lowest"
         )) %>%
  arrange(desc(ConvexArea))

Lastly for the prescriptive part of the assignment, I am basically doing what I did above but for each unique class of bean. The main difference is the use of faceting, which is one of the most recent topics we covered. The facet_wrap function lets us specify that we are making a plot for each class, and also that the y is free to take values that “make sense” if you will. This is why the different plots have different maximums on the y-axis.

ggplot(new_beans_data, aes(x = Extent)) +
  geom_histogram(aes(y = after_stat(density)), binwidth = 0.01, fill = "blue", alpha = 0.7) +
  geom_density(color = "red", linewidth = 1) +
  facet_wrap(~ Class, scales = "free_y") +
  labs(title = "Distribution of Bean Extent by Class",
       x = "Extent",
       y = "Density")

By looking at the scatter plot, we notice it is hard to see any sort of correlation or relationship. For example, somewhere between 3 < pH < 3.5, there is a pH that has every wine quality. And just by looking at the overall structure of the points, it is clear there is no real relationship. The correlation value we calculated backs the claims we can come up with from the scatterplot, and we can conclude there is no significant relationship between pH and wine quality.
